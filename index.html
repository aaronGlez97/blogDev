¿Programar por donde empiezo?

Programar es el arte de escribir sistemas de escritorio, apps web, apps mobiles , videojuegos que las personas usan en su dia a dia.

Eso es basicamente lo que hace un programador .

Pues existen distintas areas de la programacion los que desarrollan a nivel web, mobile, videojuegos.

Programador web usa las siguientes herramientas:

Basicamente utiliza los siguientes lenguajes de etiquetado,programacion

Para desarrollar la interfaz web:

HTML5 (lenguaje de etiquetado) , CSS3 , JS, Frameworks CSS los cuales pueden ser Bootstrap, Foundation, Materialize

Para desarrollar el back-end :

Se usa un lenguaje del lado del servidor como Java, Python, Ruby, Nodejs , PHP y una base de datos como Mysql o Postgresql, etc . Todo esto para programar una app compleja que necesite login , manipulacion de datos etc.

Pogramador mobile puedes ser android dev o ios dev depende del sistema movil que te guste se utilizan los siguientes lenguajes

Para programar en Android se programa en Java o Kotlin

Para programar en IOS se programa en OBJETIVE-C o Swift

Programador de videojuegos : Principalmente programan en C# o C++ y se usa Unreal-Engine o Unity

Esas son una de las areas mas importantes del desarrollo, si quieres especializarte en una ya sabes un poco lo que tienes que aprender
